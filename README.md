# Set environment 

## client

```bash
npm install
```

## flask-backend (set python virtual environment)

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

```bash
pip install Flask
```

# Start Environment

## start client (on localhost:3000)

```bash
npm start
```

## start flask-backend (on localhost:5000)

```bash
python3 server.py
```
